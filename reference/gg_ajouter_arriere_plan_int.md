# Ajouter un arrière plan de classes à un ggplot

Ajouter un arrière plan de classes à un ggplot

## Usage

``` r
gg_ajouter_arriere_plan_int(graphique, df_classes)
```

## Arguments

- graphique:

  Graphique ggplot.

- df_classes:

  Dataframe contenant les limites de classes comme classe_ipr.

## Value

Le graphique complété par son arrière-plan coloré.

## Examples

``` r
if (FALSE) { # \dontrun{
ggplot(data = data) %>%
  gg_ajouter_arriere_plan_int()
} # }
```
