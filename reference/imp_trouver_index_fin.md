# Trouver la ligne de fin qui correspond à une table désignée par sa ligne de début

Par exemple pour trouver le numéro de la ligne de fin d'une table qui
débuterait à la ligne 6517552

## Usage

``` r
imp_trouver_index_fin(vecteur_index_lignes_fin, ligne_debut)
```

## Arguments

- vecteur_index_lignes_fin:

  Vecteur contenant les numéros de toutes les lignes contenant la
  séquence caractéristique de la fin d'une table.

- ligne_debut:

  Numéro de la ligne de début de la table.

## Value

Le numéro de la ligne de fin de la table.

## Examples

``` r
if (FALSE) { # \dontrun{
imp_trouver_index_fin(vecteur_index_lignes_fin = mon_vecteur, ligne_debut = 6517552)
} # }
```
