# Ajouter à la passerelle les intervenants sur les opérations Ces intervenants sont l'opérateur, le commanditaire et le valideur

Ajouter à la passerelle les intervenants sur les opérations Ces
intervenants sont l'opérateur, le commanditaire et le valideur

## Usage

``` r
mef_ajouter_intervenants(df)
```

## Arguments

- df:

  Dataframe "passerelle" mettant en correspondance les identifiants des
  différentes tables.

## Value

La passerelle complétée.

## Examples

``` r
if (FALSE) { # \dontrun{
passerelle <- mef_creer_passerelle() %>%
mef_ajouter_intervenants()
} # }
```
