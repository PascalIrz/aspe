# Extraire les noms des tables à partir des lignes du dump

Extraire les noms des tables à partir des lignes du dump

## Usage

``` r
imp_extraire_noms_tables(lignes_dump)
```

## Arguments

- lignes_dump:

  Nom de la liste qui contient les lignes du dump.

## Value

Un vecteur caractères contenant les noms des tables.

## Examples

``` r
if (FALSE) { # \dontrun{
imp_extraire_noms_tables (lignes_dump = mes_lignes_du_dump)
} # }

```
