# Rajouter le libellé du site

**\[deprecated\]**. Fonction remplacée par "mef_ajouter_libelle_site()".

## Usage

``` r
mef_ajouter_libelle(passerelle)
```

## Arguments

- passerelle:

  Dataframe "passerelle" mettant en correspondance les identifiants des
  différentes tables.

## Value

La passerelle à laquelle ont été ajoutés les libellés des points de
prélèvement.

## Examples

``` r
if (FALSE) { # \dontrun{
passerelle <- mef_ajouter_libelle(passerelle = passerelle)
} # }
```
