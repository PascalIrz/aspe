# Ajouter les notes IPR à une table passerelle

Ajouter les notes IPR à une table passerelle

## Usage

``` r
mef_ajouter_ipr(passerelle)
```

## Arguments

- passerelle:

  Dataframe mettant en correspondance les identifiants des différentes
  tables.

## Value

Dataframe avec la valeur de l'IPR, et la classe de qualité.

## Examples

``` r
if (FALSE) { # \dontrun{
mon_df_ipr <- mef_ajouter_ipr(passerelle = passerelle)
} # }
```
