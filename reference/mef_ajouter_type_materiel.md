# Ajouter à la passerelle le type de matériel employé

ex : Héron

## Usage

``` r
mef_ajouter_type_materiel(df)
```

## Arguments

- df:

  Dataframe "passerelle" mettant en correspondance les identifiants des
  différentes tables.

## Value

La passerelle complétée.

## Examples

``` r
if (FALSE) { # \dontrun{
passerelle <- passerelle %>%
mef_ajouter_type_materiel()
} # }
```
