# Ajouter à la passerelle l'a qualification de a donnée'état d'avancement de la saisie

ex : En cours de saisie / Validé niveau 1.

## Usage

``` r
mef_ajouter_validation(df)
```

## Arguments

- df:

  Dataframe "passerelle" mettant en correspondance les identifiants des
  différentes tables.

## Value

La passerelle complétée.

## Examples

``` r
if (FALSE) { # \dontrun{
passerelle <- passerelle %>%
mef_ajouter_validation()
} # }
```
